# ⛽🇩🇪 Fuel Widget for Android (Tankstelle)

An adaptive Android widget that finds the nearest gas stations based on your location and displays current fuel prices.

[English](#english) | [Русский](#русский) | [Українська](#українська) | [Deutsch](#deutsch)

## English <a name="english"></a>
### Features
* **Automatic Location:** Finds the nearest gas stations using GPS.
* **Wide Fuel Selection:** Supports 15 types of fuel (Diesel, Super E5/E10, LPG, LNG, HVO, etc.).
* **Adaptive Design:** Automatically adjusts content for 2, 3, 4, or 5 column widgets.
* **Visual Indicators:** Color-coded city names and price update times.

## Русский <a name="русский"></a>
### Описание
Адаптивный виджет для Android, который находит ближайшие заправочные станции в зависимости от вашего местоположения и отображает актуальные цены.

### Функции
* **Автоматический поиск:** Определяет координаты и ищет заправки рядом.
* **Огромный выбор топлива:** Поддержка 15 видов (Diesel, Super E5/E10, LPG, LNG, HVO Diesel, AdBlue и др.).
* **Умный интерфейс:** Названия городов выделены цветом, а время обновления цены автоматически сокращается для экономии места.
* **Адаптивность:** Корректно отображается при растягивании виджета от 2 до 5 колонок.

## Українська <a name="українська"></a>
### Опис
Адаптивний віджет для Android, який знаходить найближчі заправні станції за вашим місцезнаходженням та відображає актуальні ціни на пальне.

### Особливості
* **Геолокація:** Використовує GPS для пошуку найближчих АЗС.
* **Багато видів пального:** Підтримка 15 типів палива (від стандартного Diesel до водню та AdBlue).
* **Зручний дизайн:** Кольорове кодування міст та інтелектуальне відображення часу оновлення.
* **Гнучкість:** Підтримка різної ширини віджета (2-5 колонок).

## Deutsch <a name="deutsch"></a>
### Beschreibung
Ein adaptives Android-Widget, das die nächstgelegenen Tankstellen basierend auf Ihrem Standort findet und aktuelle Kraftstoffpreise anzeigt.

### Hauptmerkmale
* **Standortbasiert:** Automatische Suche nach Tankstellen in der Nähe via GPS.
* **Große Kraftstoffauswahl:** Unterstützt 15 Sorten (Diesel, Super E5/E10, LPG, LNG, HVO, Wasserstoff usw.).
* **Intelligentes Design:** Farblich hervorgehobene Ortsnamen und platzsparende Datumsanzeige.
* **Anpassbar:** Optimiert für Widget-Breiten von 2 bis 5 Spalten.

---
